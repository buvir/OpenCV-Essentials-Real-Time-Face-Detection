# Real-Time Face Detection using OpenCV + Haar Cascades

## Overview
A simple Python project that uses OpenCV to detect faces in real-time using your webcam. Bounding boxes are drawn around detected faces.

## Features
- Real-time face detection
- Uses Haar Cascade classifier (classical computer vision)
- Demonstrates grayscale conversion, feature extraction, and sliding window detection

How to Run
```
python face_detection.py

```
